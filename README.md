# O-intecOptima - 光模块智能运维平台

O-intecOptima是一个专业的光模块智能运维平台，集成了光模块监控、预测分析、智能运维等功能，帮助用户更好地管理和维护光模块设备。

## 版本历史

### v0.8 (2025.03.24)
#### 新特性
- 📊 仪表盘功能增强
  - 支持多设备选择和管理
  - 添加详细的错误统计（丢包、CRC错误、帧错误）
  - 集成SONiC网络接口信息展示
  - 支持gNMI telemetry数据采集和展示
- 🎯 预测规则管理系统升级
  - 新增行业场景规则支持
  - 优化预测算法配置
  - 添加规则导出和历史趋势分析
  - 支持自定义预测模型和参数
- 💡 用户体验优化
  - 优化页面布局和交互设计
  - 增强数据可视化效果
  - 改进错误提示和帮助信息
  - 提升系统响应速度

## 技术栈
- 前端：React + TypeScript + Vite
- UI框架：TailwindCSS
- 图表：D3.js
- 状态管理：React Hooks
- 开发工具：ESLint + Prettier

## 开发指南

### 启动开发服务器

# O-intecOptima - 光模块智能运维平台

O-intecOptima是一个专业的光模块智能运维平台，集成了光模块监控、预测分析、智能运维等功能，帮助用户更好地管理和维护光模块设备。

## 版本历史

#### 新特性
- 🎯 增强光模块预测分析功能

  - 添加预警模块列表和状态监控
  - 集成历史趋势分析功能
- 🤖 集成LLM智能助手服务
  - 新增本地LLM服务支持
  - 添加API接口和类型定义
  - 实现useLLM自定义Hook
  - 支持多模型配置和切换
- 🛠️ 技术架构优化
  - 更新路由配置，优化页面导航

  - 优化项目结构和代码组织

## 技术栈
- 前端：React + TypeScript + Vite
- UI框架：TailwindCSS
- 图表：D3.js
- 状态管理：React Hooks
- 开发工具：ESLint + Prettier

## 开发指南
### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 启动LLM服务器
```bash
cd src/services/llm/server
npm install
npm start
```
